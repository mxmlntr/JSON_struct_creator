Detailed Description

main of the JSON_struct_creator

This project is for generating JSON files and the neccessary code (struct, archive, CRC, parse) to run it. The size can be predefined by the number of cycles. ATTENTION: These generated JSON files are with the only purpose of simulating different sizes JSON files, they depend on a repetitive algorithm.
